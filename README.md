 🚚 Transport & Logistics Dashboard | Power BI

##  Project Overview
This project is a Transport & Logistics Analysis Dashboard created using **Power BI**.

The dashboard helps to analyze shipment performance, delivery status, suppliers, customers, vehicles, and drivers.

##  Tools Used
- Power BI
- Excel
- DAX
- Data Visualization

## 📊 Dashboard Features

### Executive Overview
- Total Deliveries
- On Time Deliveries
- Delayed Deliveries
- Shipment Analysis
- Customer & Supplier Performance

### Delivery Performance
- Delivery trend analysis
- On-time vs delayed delivery
- Vehicle-wise delivery analysis
- Location-based shipment tracking

### Supplier & Customer Analysis
- Top Customers
- Top Suppliers
- Material shipped analysis
- Driver performance

##  Key KPIs
- Total Shipments
- Delivery Status
- On Time %
- Delayed %

##  Dataset
The dataset contains:
- Shipment details
- Customer information
- Supplier details
- Vehicle type
- Driver information
- GPS location
- Delivery status
